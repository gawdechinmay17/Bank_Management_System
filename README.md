# Bank_Management_System
Designed and developed a fully functional bank management system in Python applying core OOP concepts — classes, inheritance, encapsulation, and polymorphism
Implemented features including account creation, deposit, withdrawal, balance inquiry, and transaction history
Used JSON for lightweight persistent data storage, enabling data read/write operations without a relational database
Built a separate interactive web-based interface using Streamlit, improving usability and user experience
Applied modular code structure separating business logic, data layer, and UI for clean and maintainable code
